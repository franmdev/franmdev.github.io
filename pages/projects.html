B2B Lead Intelligence Pipeline

Data Engineering · MLOps · NLP · Infraestructura · ChatOps

Pipeline empresarial de 6 fases con arquitectura Medallion (Bronze → Silver → Gold) para extracción, procesamiento y clasificación automática de leads B2B. Orquestado via Telegram Bot (ChatOps RBAC) desde cualquier dispositivo, desplegado en Proxmox VE con LXC air-gapped y sincronización Zero-Trust a la nube. La capa Gold clasifica ofertas con BETO (Spanish BERT) fine-tuned y loop de Active Learning.

  • F1 Command & Control: Telegram Bot con RBAC, telemetría en tiempo real desde status.json, IPC por flags + os.fsync() para graceful shutdown Zero-Trust, capturas forenses con PyAutoGUI.
  • F2 Extracción Biomimética: Selenium + Microsoft Edge con perfiles persistentes. Session hijacking prevention, delays gaussianos anti-detección bot, self-healing ante login walls con alerta HITL automática.
  • F3 Bronze Layer: Ingesta event-driven con spooler asíncrono. Limpieza NLP (emojis, HTML, URLs, truncamientos SPA), merge histórico incremental y schema enforcement con pandas.
  • F4 High Availability Zero-Trust: Solución al kernel Linux Estado D en Proxmox. Defensa en 2 capas: IPC elegante + hook Perl desfibrilador (umount -f -l). rclone a OneDrive/S3 con validación MD5.
  • F5 Silver Layer MDM: Pipeline 5 fases — hash exacto (xxhash), filtro heurístico regex, near-duplicates CUDA/MinHash, verificación Levenshtein. Voting ensemble 5 capas para filtro de idioma español.
  • F6 Gold Layer MLOps: BETO fine-tuning con class weights automáticos para dataset desbalanceado, FP16 mixed precision, Active Learning con triage de 5 buckets exportado a Excel con formato condicional dinámico para QA.
