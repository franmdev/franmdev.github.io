Este Portafolio: Full Stack Data System

Arquitectura Web | Seguridad | Data Pipeline

Problema

Construir un portafolio que no fuera solo una "vitrina", sino un proyecto demostrable de arquitectura web full-stack, seguridad y data engineering.

Solución

Sistema completo con:

  • Frontend: GitHub Pages (estático, CDN global)
  • Backend: Azure Functions (serverless Python)
  • Base de datos: PostgreSQL en Azure (analytics en tiempo real)
  • Seguridad: Defensa en 5 capas (Cloudflare WAF, Turnstile, ipapi.is, Rate Limit, Caché DB)
  • Flujo dinámico: Validación inteligente con "Fast Pass" para usuarios de confianza

Resultados

  • ✅ Seguridad: 99.9% bloqueo de tráfico malicioso
  • ✅ Performance: <200ms response time
  • ✅ Costo: Optimizado a tier gratuito (B1ms PostgreSQL)
  • ✅ Analytics: Recolección en tiempo real de datos de navegación
